# brand

Logo marks, favicon source, and Open Graph / Twitter share image.

| File | Purpose |
|---|---|
| `logo.svg` | Primary wordmark/logomark, used in the navbar and footer |
| `og-image.jpg` | 1200×630 social share image, referenced from `metadata.openGraph.images` in `src/app/layout.tsx` |
| `favicon-source.svg` | Source file for `src/app/favicon.ico` |

None of these exist yet — the site currently uses a text wordmark and the
default Next.js favicon as placeholders.
