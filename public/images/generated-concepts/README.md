# generated-concepts

AI-generated or exploratory visuals only — moodboards, style directions,
speculative UI concepts not tied to delivered client work.

Rules:

- Every image used from this folder must be labeled **"Concept"** in the UI
  (the `ImagePlaceholder` component does this automatically for any image
  without a confirmed `src`, and any concept image should keep an explicit
  "Concept" label even after a real file is added).
- Never present a concept visual as completed client work, a result, or a
  testimonial.
- Do not use these images on a project's Dashboard/Mobile/Workflow slots once
  real product screenshots exist — replace them instead of layering both.
